项目定位

OpenClaw 是一个自托管的 AI 网关，它连接你的聊天应用（WhatsApp、Telegram、Discord 等）与 AI 助手。

核心特性

  • 完全自托管：数据完全自主掌控
  • 多渠道支持：一个网关同时服务多个平台
  • 原生 Agent 支持：内置记忆、工具调用
  • 开源免费：MIT 许可

快速开始

npm install -g openclaw@latest
openclaw gateway --port 18789
